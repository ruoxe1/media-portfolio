# 覃勇波数字媒体作品集

这是一个用于实习投递的静态作品集网站，入口是 `index.html`。

## 使用方式

- 直接打开 `index.html` 预览。
- 作品视频放在 `assets/videos/`。
- 作品信息集中在 `app.js` 的 `works` 配置中，后续改标题、职责、视频路径都在这里处理。
- 页面里的“添加作品”会把新作品保存在当前浏览器本地，不会上传文件；换电脑或清理浏览器数据后需要重新添加。
